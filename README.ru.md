# 🎮 QuestFlow

[🇬🇧 English](README.md)

> **Агрегатор игровых цен с поддержкой региональных прайсов Steam и уведомлениями в Telegram**

---

## 📝 Описание проекта

**QuestFlow** — это современная платформа для геймеров, позволяющая отслеживать цены на игры, сравнивать предложения в различных регионах Steam и получать мгновенные уведомления о скидках. Проект разработан как веб-приложение с поддержкой PWA и интеграцией с Telegram, обеспечивая максимальное удобство использования как на десктопе, так и на мобильных устройствах.

## 🛠 Стек технологий

Проект использует современный и производительный стек:

*   **Frontend**: React, TypeScript, Vite
*   **Управление состоянием**: Zustand
*   **Запросы к данным**: TanStack Query (React Query)
*   **Стилизация**: CSS / Glassmorphism
*   **Бэкенд / Бот**: Python (для бэкенда/бота)
*   **Автоматизация**: GitHub Actions
*   **Хостинг**: Firebase

## ✨ Ключевые фичи (Highlights)

*   🌍 **Региональные цены**: Умная логика пересчета и отображения цен для различных регионов Steam (например, KZT, RUB, USD), помогающая находить самые выгодные предложения.
*   📱 **PWA (Progressive Web App)**: Возможность полноценной установки приложения на десктоп и мобильные устройства, обеспечивающая нативный опыт.
*   🤖 **Telegram Mini App + Бот**: Доступ к платформе прямо из мессенджера Telegram и бот для своевременных уведомлений о падении цен.
*   ⏰ **Автоматизация через GitHub Actions**: Фоновая проверка цен каждые 6 часов гарантирует актуальность информации без участия пользователя.
